# 🚀 AVIATOR AI PRO

Scanner de Velas Rosas com Neuroplasticidade Integrada

## Funcionalidades

- 📊 **Dashboard** — Métricas em tempo real
- 🎯 **Feed ao Vivo** — Últimas 50 velas
- 🧠 **Neuroplasticidade** — 5 mecanismos de aprendizado
- 🧮 **Calculadora** — Cálculo de ganhos
- 📋 **Catálogo** — Tabela de sinais

## Instalação Local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy no Streamlit Cloud

1. Faça push para GitHub
2. Acesse https://share.streamlit.io/
3. Clique em "New app"
4. Selecione seu repositório
5. Deploy automático!
